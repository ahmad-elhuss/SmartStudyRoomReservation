# Smart Study Room Reservation System

## Project Description
A JavaFX desktop application for reserving study rooms in the university. Users can register, login, view available rooms, make reservations, and manage their bookings.

## Features
- User authentication (Login/Register)
- Real-time room availability
- Reservation management
- Admin panel for room management
- Client-Server architecture using TCP Sockets
- Multithreading support

## Technologies
- Java + JavaFX
- Scene Builder
- TCP Sockets
- OOP Principles (Encapsulation, Polymorphism, Inheritance)

## How to Run
1. Run the Server first (`Server.java`)
2. Run the Client (`ClientApplication.java`)
